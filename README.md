# Rent Split Calculator

A Python-based CLI tool designed to help students and bachelors living in shared accommodations easily calculate and split monthly expenses.

# Features

- **Comprehensive Calculation:** Adds up Rent, Food, Electricity, Water, Grocery, and Maintenance costs.
- **Smart Logic:** Automatically calculates electricity bills based on units consumed and rate per unit.
- **Fair Split:** Divides the total equally among the number of roommates.
- **Error Handling:** Prevents crashes if users enter invalid text or zero roommates.

# Requirements

- Python 3.x
